# Древо процессов

**Цель**: Реализовать программу порождающую процессы по схеме. Каждый процесс выводит свои `pid` и `ppid`.

**Схема процессов**:
```
Программа
├── Процесс_1
│   ├── Процесс_3
│   └── Процесс_4
└── Процесс_2
    └── Процесс_5
```

**Для запуска**:
- Скомпилировать файл `src/main.c` с помощью `gcc`
- Запустить бинарный файл
